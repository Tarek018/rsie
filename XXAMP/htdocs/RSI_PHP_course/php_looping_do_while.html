Do While Loop

PHP do while Loop

The do...while loop - Loops through a block of code once, and then repeats the loop as long as the specified condition is true.

Boucle PHP do while

La boucle do...while - Boucles à travers un bloc de code une fois, puis répète la boucle tant que le spécifié la condition est vraie.

The PHP do...while Loop

The do...while loop will always execute the block of code once, it will then check the condition, and repeat the loop while the specified condition is true.

Syntax

La boucle PHP do...while

La boucle do...while exécutera toujours le bloc de code une fois, il vérifiera alors la condition et répétera la boucle tant que la condition spécifiée est vraie.

Syntaxe

do {
code to be executed;
}
while (condition is true );

Examples

The example below first sets a variable $x to 1 ($x = 1). Then, the do while loop will write some output, and then increment the variable $x with 1. Then the condition is checked (is $x less than, or equal to 5?), and the loop will continue to run as long as $x is less than, or equal to 5:

Exemples

L'exemple ci-dessous définit d'abord une variable $x sur 1 ($x = 1). Ensuite, le faire pendant loop écrira une sortie, puis incrémentera la variable $x avec 1. Ensuite, la condition est vérifiée (est $x inférieur ou égal à 5 ?), et la boucle continuera à s'exécuter tant que $x est inférieur ou égal à 5 :

Example

 <?php 
 $x = 1;
 do {
 echo "The number is: $x <br>";
 $x++;
} while ($x <= 5);
?>

Note: In a do...while loop the condition is tested AFTER executing the statements within the loop. This means that the do...while loop will execute its statements at least once, even if the condition is false. See example below.

This example sets the $x variable to 6, then it runs the loop, and then the condition is checked:

Remarque : dans une boucle do...while la condition est testée APRÈS l'exécution les instructions dans la boucle. Cela signifie que la boucle do...while s'exécutera ses déclarations au moins une fois, même si la condition est fausse. Voir l'exemple ci-dessous.

Cet exemple définit la variable $x sur 6, puis il exécute la boucle, puis la condition est vérifiée :

Example

 <?php 
 $x = 6;
do {
 echo "The number is: $x <br>";
$x++;
} while ($x <= 5);


 ?>
