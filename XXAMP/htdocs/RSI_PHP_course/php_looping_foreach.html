PHP foreach Loop

The foreach loop - Loops through a block of code for each element in an array.

Boucle PHP foreach

La boucle foreach - Boucles via un bloc de code pour chaque élément d'un tableau.

The PHP foreach Loop

The foreach loop works only on arrays, and is used to loop through each key/value pair in an array.

Syntax

La boucle PHP foreach

La boucle foreach ne fonctionne que sur les tableaux et est utilisée pour parcourir chaque paire clé/valeur d'un tableau.

Syntaxe

foreach ($array as $value) {
code to be executed;
}

For every loop iteration, the value of the current array element is assigned to $value and the array pointer is moved by one, until it reaches the last array element.

Examples

The following example will output the values of the given array ($colors):

Pour chaque itération de boucle, la valeur de l'élément courant du tableau est assigné à $value et le pointeur du tableau est déplacé de un, jusqu'à ce qu'il atteigne le dernier élément du tableau.

Exemples

L'exemple suivant affichera les valeurs des tableau donné ($colors):

Example

<?php
$colors = array("red", "green", "blue", "yellow"); 

foreach ($colors as $value) {
 echo "$value <br>";
}
 ?>

The following example will output both the keys and the values of the given array ($age):

L'exemple suivant affichera à la fois les clés et les valeurs des tableau donné ($age):

Example

 <?php
$age = array("Samir"=>"35", "Salim"=>"37", "Yassine"=>"43");
foreach($age as $x => $val) {
 echo "$x = $val<br>";
}

?>
