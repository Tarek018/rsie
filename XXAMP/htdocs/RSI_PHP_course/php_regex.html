PHP Regular Expressions

What is a Regular Expression?

A regular expression is a sequence of characters that forms a search pattern. When you search for data in a text, you can use this search pattern to describe what you are searching for.

A regular expression can be a single character, or a more complicated pattern.

Regular expressions can be used to perform all types of text search and text replace operations.

PHP Expressions régulières

Qu'est-ce qu'une expression régulière ?

Une expression régulière est une séquence de caractères qui forme un modèle de recherche. Lorsque vous recherchez des données dans un texte, vous pouvez utiliser ce modèle de recherche pour décrire ce que vous recherchez.

Une expression régulière peut être un caractère unique ou un modèle plus compliqué.

Les expressions régulières peuvent être utilisées pour effectuer tous les types de recherche de texte et de remplacement de texte opérations.

Syntax

In PHP, regular expressions are strings composed of delimiters, a pattern and optional modifiers.

Syntaxe

En PHP, les expressions régulières sont des chaînes composées de délimiteurs, d'un motif et de modificateurs.

$exp = "/tebessa-univ/i";

In the example above, / is the delimiter, tebessa-univ is the pattern that is being searched for, and i is a modifier that makes the search case-insensitive.

The delimiter can be any character that is not a letter, number, backslash or space. The most common delimiter is the forward slash (/), but when your pattern contains forward slashes it is convenient to choose other delimiters such as # or ~.

Dans l'exemple ci-dessus, / est le délimiteur, tebessa-univ est le modèle qui est recherché, et i est un modificateur qui rend la recherche insensible à la casse.

Le délimiteur peut être n'importe quel caractère autre qu'une lettre, un chiffre, une barre oblique inverse ou un espace. La le délimiteur le plus courant est la barre oblique (/), mais lorsque votre modèle contient barres obliques, il est pratique de choisir d'autres délimiteurs tels que # ou ~.

Regular Expression Functions

PHP provides a variety of functions that allow you to use regular expressions. The preg_match(), preg_match_all() and preg_replace() functions are some of the most commonly used ones:

Function Description
preg_match() Returns 1 if the pattern was found in the string and 0 if not
preg_match_all() Returns the number of times the pattern was found in the string, which may also be 0
preg_replace() Returns a new string where matched patterns have been replaced with another string

Fonctions d'expression régulière

PHP fournit une variété de fonctions qui vous permettent d'utiliser des expressions régulières. La preg_match(), preg_match_all() et preg_replace() Les fonctions font partie des les plus couramment utilisés :

Fonction Description
preg_match() Renvoie 1 si le motif a été trouvé dans la chaîne et 0 sinon
preg_match_all() Renvoie le nombre de fois que le motif a été trouvé dans la chaîne, qui peut aussi être 0
preg_replace() Renvoie une nouvelle chaîne où les modèles correspondants ont été remplacés par une autre chaîne

Using preg_match()

The preg_match() function will tell you whether a string contains matches of a pattern.

Utiliser preg_match()

La fonction preg_match() vous dira si une chaîne contient des correspondances d'un modèle.

Example

Use a regular expression to do a case-insensitive search for "tebessa-univ" in a string:

 <?php
$str = "Visit tebessa-univ";
$pattern = "/tebessa-univ/i";
echo preg_match($pattern, $str); // Outputs 1
?>

Using preg_match_all()

The preg_match_all() function will tell you how many matches were found for a pattern in a string.

Utiliser preg_match_all()

La fonction preg_match_all() vous indiquera combien de correspondances ont été trouvées pour un modèle dans un chaîne.

Example

Use a regular expression to do a case-insensitive count of the number of occurrences of "ain" in a string:

 <?php
$str = "The rain in SPAIN falls mainly on the plains.";
$pattern = "/ain/i";
echo preg_match_all($pattern, $str); // Outputs 4
?>

Using preg_replace()

The preg_replace() function will replace all of the matches of the pattern in a string with another string.

Utiliser preg_replace()

La fonction preg_replace() remplacera toutes les correspondances du modèle dans une chaîne par une autre chaîne.

Example

Use a case-insensitive regular expression to replace Microsoft with tebessa-univ in a string:

 <?php

$str = "Visit Microsoft!";
$pattern = "/microsoft/i";
echo preg_replace($pattern, "tebessa-univ", $str); 
// Outputs "Visit tebessa-univ!"


?>

Regular Expression Modifiers

Modifiers can change how a search is performed.

Modifier Description
i Performs a case-insensitive search
m Performs a multiline search (patterns that search for the beginning or end of a string will match the beginning or end of each line)
u Enables correct matching of UTF-8 encoded patterns

Modificateurs d'expressions régulières

Les modificateurs peuvent changer la façon dont une recherche est effectuée.

Modificateur Description
je Effectue une recherche insensible à la casse
m Effectue une recherche multiligne (modèles qui recherchent le le début ou la fin d'une chaîne correspondra au début ou à la fin de chaque ligne)
u Permet une correspondance correcte des modèles encodés en UTF-8

Regular Expression Patterns

Brackets are used to find a range of characters:

Expression Description
[abc] Find one character from the options between the brackets
[^abc] Find any character NOT between the brackets
[0-9] Find one character from the range 0 to 9

Metacharacters

Metacharacters are characters with a special meaning:

Metacharacter Description
| Find a match for any one of the patterns separated by | as in: cat|dog|fish
. Find just one instance of any character
^ Finds a match as the beginning of a string as in: ^Hello
$ Finds a match at the end of the string as in: World$
\d Find a digit
\s Find a whitespace character
\b Find a match at the beginning of a word like this: \bWORD, or at the end of a word like this: WORD\b
\uxxxx Find the Unicode character specified by the hexadecimal number xxxx

Quantifiers

Quantifiers define quantities:

Quantifier Description
n+ Matches any string that contains at least one n
n* Matches any string that contains zero or more occurrences of n
n? Matches any string that contains zero or one occurrences of n
n{x} Matches any string that contains a sequence of X n's
n{x,y} Matches any string that contains a sequence of X to Y n's
n{x,} Matches any string that contains a sequence of at least X n's

Note: If your expression needs to search for one of the special characters you can use a backslash ( \ ) to escape them. For example, to search for one or more question marks you can use the following expression: $pattern = '/\?+/';

Modèles d'expressions régulières

Les crochets sont utilisés pour trouver une plage de caractères :

Expression Description
[abc] Trouvez un caractère parmi les options entre crochets
[^abc] Rechercher tout caractère NON entre crochets
[0-9] Trouvez un caractère entre 0 et 9

Métacaractères

Les métacaractères sont des caractères ayant une signification particulière :

Métacaractère Description
| Trouvez une correspondance pour l'un des motifs séparés par | comme dans : chat|chien|poisson
. Trouvez une seule instance de n'importe quel caractère
^ Trouve une correspondance comme début d'une chaîne comme dans : ^Hello
$ Trouve une correspondance à la fin de la chaîne comme dans : World$
\d Rechercher un chiffre
\s Rechercher un caractère d'espace blanc
\b Trouvez une correspondance au début d'un mot comme ceci : \bWORD, ou à la fin d'un mot comme ceci : WORD\b
\uxxxx Recherchez le caractère Unicode spécifié par le nombre hexadécimal xxxx

Quantificateurs

Les quantificateurs définissent les quantités :

Quantificateur Description
n+ Correspond à toute chaîne contenant au moins un n
n* Reconnaît n'importe quelle chaîne qui contient zéro ou plusieurs occurrences de n
n ? Reconnaît n'importe quelle chaîne contenant zéro ou une occurrence de n
n{x} Reconnaît toute chaîne contenant une séquence de X n
n{x,y} Reconnaît n'importe quelle chaîne contenant une séquence de X à Y n
n{x,} Reconnaît toute chaîne contenant une séquence d'au moins X n

Remarque : Si votre expression doit rechercher l'un des caractères spéciaux, vous pouvez utiliser un barre oblique inverse ( \ ) pour les échapper. Par exemple, pour rechercher un ou plusieurs points d'interrogation, vous pouvez utiliser les éléments suivants expression : $motif = '/\?+/';

Grouping

You can use parentheses ( ) to apply quantifiers to entire patterns. They also can be used to select parts of the pattern to be used as a match.

Regroupement

Vous pouvez utiliser des parenthèses ( ) pour appliquer des quantificateurs à des motifs entiers. Ils peuvent également être utilisés pour sélectionner des parties du motif à utiliser comme correspondance.

Example

Use grouping to search for the word "banana" by looking for ba followed by two instances of na:

<?php
$str = "Apples and bananas.";
$pattern = "/ba(na){2}/i";
echo preg_match($pattern, $str); // Outputs 1
?>
