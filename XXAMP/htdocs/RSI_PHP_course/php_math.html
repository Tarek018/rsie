Math

PHP Math

PHP has a set of math functions that allows you to perform mathematical tasks on numbers.

Mathématiques PHP

PHP dispose d'un ensemble de fonctions mathématiques qui vous permettent d'effectuer des tâches mathématiques sur des nombres.

PHP pi() Function

The pi() function returns the value of PI:

Fonction PHP pi()

La fonction pi() renvoie la valeur de PI :

Example

<?php
echo(pi()); // returns 3.1415926535898
?>

PHP min() and max() Functions

The min() and max() functions can be used to find the lowest or highest value in a list of arguments:

Fonctions PHP min() et max()

Les fonctions min() et max() peuvent être utilisées pour trouver la valeur la plus basse ou la plus haute dans une liste d'arguments :

Example

<?php
echo(min(0, 150, 30, 20, -8, -200)); // returns -200
echo "<br>";
echo(max(0, 150, 30, 20, -8, -200)); // returns 150
?>

PHP abs() Function

The abs() function returns the absolute (positive) value of a number:

Fonction PHP abs()

La fonction abs() renvoie la valeur absolue (positive) d'un nombre :

Example

<?php
echo(abs(-6.7)); // returns 6.7
?>

PHP sqrt() Function

The sqrt() function returns the square root of a number:

Fonction PHP sqrt()

La fonction sqrt() renvoie la racine carrée d'un nombre :

Example

<?php
echo(sqrt(64)); // returns 8
?>

PHP round() Function

The round() function rounds a floating-point number to its nearest integer:

Fonction PHP round()

La fonction round() arrondit un nombre à virgule flottante à son entier le plus proche :

Example

  <?php
 echo(round(0.60)); // returns 1
 echo "<br>";
 echo(round(0.49)); // returns 0
?>

Random Numbers

The rand() function generates a random number:

Nombres aléatoires

La fonction rand() génère un nombre aléatoire :

Example

<?php
echo(rand());
?>

To get more control over the random number, you can add the optional min and max parameters to specify the lowest integer and the highest integer to be returned.

For example, if you want a random integer between 10 and 100 (inclusive), use rand(10, 100):

Pour mieux contrôler le nombre aléatoire, vous pouvez ajouter les paramètres facultatifs min et max pour spécifier l'entier le plus bas et l'entier le plus élevé à renvoyer.

Par exemple, si vous voulez un entier aléatoire entre 10 et 100 (inclus), utilisez rand(10, 100) :

Example

<?php
echo(rand(10, 100));
?>
