This chapter shows how to keep the values in the input fields when the user hits the submit button.

PHP Exemple de formulaire complet

Ce chapitre montre comment conserver les valeurs dans les champs de saisie lorsque l'utilisateur appuie sur le bouton d'envoi.

PHP - Keep The Values in The Form

To show the values in the input fields after the user hits the submit button, we add a little PHP script inside the value attribute of the following input fields: name, email, and website. In the comment textarea field, we put the script between the <textarea> and </textarea> tags. The little script outputs the value of the $name, $email, $website, and $comment variables.

PHP - Conserver les valeurs dans le formulaire

Pour afficher les valeurs dans les champs de saisie une fois que l'utilisateur a cliqué sur le bouton d'envoi, nous ajoutons un petit script PHP à l'intérieur de l'attribut value de l'entrée suivante champs : nom, e-mail et site Web. Dans le champ de zone de texte de commentaire, nous mettons le script entre le <textarea> et </textarea> Mots clés. Le petit script affiche le valeur des variables $name, $email, $website et $comment.

Ensuite, nous devons également montrer quel bouton radio a été coché. Pour cela, nous doit manipuler l'attribut coché (pas l'attribut de valeur pour la radio boutons):

Name: <input type="text" name="name" value="<?php echo $name;?>">
E-mail: <input type="text" name="email" value="<?php echo $email;?>">
Website: <input type="text" name="website" value="<?php echo $website;?>">
Comment: <textarea name="comment" rows="5" cols="40"><?php echo $comment;?></textarea>
Gender:
<input type="radio" name="gender" <?php if (isset($gender) && $gender=="female") echo "checked";?> value="female">Female
<input type="radio" name="gender" <?php if (isset($gender) && $gender=="male") echo "checked";?> value="male">Male
<input type="radio" name="gender" <?php if (isset($gender) && $gender=="other") echo "checked";?> value="other">Other
