PHP Date and Time

The PHP date() function is used to format a date and/or a time.

PHP Date et heure

La fonction PHP date() permet de formater une date et/ou une heure.

The PHP Date() Function

The PHP date() function formats a timestamp to a more readable date and time.

Syntax

La fonction PHP Date()

La fonction PHP date() formate un horodatage en une date et une heure plus lisibles.

Syntaxe

date(format,timestamp)
Parameter Description
format Required. Specifies the format of the timestamp
timestamp Optional. Specifies a timestamp. Default is the current date and time

A timestamp is a sequence of characters, denoting the date and/or time at which a certain event occurred.

Paramètre Description
format Obligatoire. Spécifie le format de l'horodatage
horodatage Facultatif. Spécifie un horodatage. La valeur par défaut est la date et l'heure actuelles

Un horodatage est une séquence de caractères indiquant la date et/ou l'heure à laquelle un certain événement s'est produit.

Get a Date

The required format parameter of the date() function specifies how to format the date (or time).

Here are some characters that are commonly used for dates:

  • d - Represents the day of the month (01 to 31)
  • m - Represents a month (01 to 12)
  • Y - Represents a year (in four digits)
  • l (lowercase 'L') - Represents the day of the week

Other characters, like"/", ".", or "-" can also be inserted between the characters to add additional formatting.

The example below formats today's date in three different ways:

Obtenir un rendez-vous

Le paramètre obligatoire format de la fonction date() spécifie comment formater la date (ou heure).

Voici quelques caractères couramment utilisés pour les dates :

  • d - Représente le jour du mois (01 à 31)
  • m - Représente un mois (01 à 12)
  • Y - Représente une année (en quatre chiffres)
  • l (minuscule 'L') - Représente le jour de la semaine

Autres caractères, comme "/", ".", ou "-" peut également être inséré entre le caractères pour ajouter une mise en forme supplémentaire.

L'exemple ci-dessous formate la date du jour de trois manières différentes :

Example

 <?php
 echo "Today is " . date("Y/m/d") . "<br>";
 echo "Today is " . date("Y.m.d") . "<br>";
 echo "Today is " . date("Y-m-d") . "<br>";
 echo "Today is " . date("l");
 ?>

Get a Time

Here are some characters that are commonly used for times:

  • H - 24-hour format of an hour (00 to 23)
  • h - 12-hour format of an hour with leading zeros (01 to 12)
  • i - Minutes with leading zeros (00 to 59)
  • s - Seconds with leading zeros (00 to 59)
  • a - Lowercase Ante meridiem and Post meridiem (am or pm)

The example below outputs the current time in the specified format:

Obtenir un temps

Voici quelques caractères couramment utilisés pour les heures :

  • H - Format 24 heures d'une heure (00 à 23)
  • h - format 12 heures d'une heure avec des zéros non significatifs (01 à 12)
  • i - Minutes avec des zéros non significatifs (00 à 59)
  • s - Secondes avec des zéros non significatifs (00 à 59)
  • a - Minuscule Ante meridiem et Post meridiem (am ou pm)

L'exemple ci-dessous affiche l'heure actuelle dans le format spécifié :

Example

 <?php
 echo "The time is " . date("h:i:sa");
 ?>

Note that the PHP date() function will return the current date/time of the server!

Notez que la fonction PHP date() renverra la date/heure actuelle du serveur !

Get Your Time Zone

If the time you got back from the code is not correct, it's probably because your server is in another country or set up for a different timezone.

So, if you need the time to be correct according to a specific location, you can set the timezone you want to use.

The example below sets the timezone to "America/New_York", then outputs the current time in the specified format:

Obtenez votre fuseau horaire

Si l'heure à laquelle vous êtes revenu du code n'est pas correcte, c'est probablement parce que votre serveur est dans un autre pays ou configuré pour un fuseau horaire différent.

Donc, si vous avez besoin de temps pour être correct en fonction d'un emplacement, vous pouvez définir le fuseau horaire que vous souhaitez utiliser.

L'exemple ci-dessous définit le fuseau horaire sur "Amérique/New_York", puis affiche l'heure actuelle dans le format spécifié :

Example

 <?php
 date_default_timezone_set("Africa/Algiers");
 echo "The time is " . date("h:i:sa");
 ?>

Create a Date With mktime()

The optional timestamp parameter in the date() function specifies a timestamp. If omitted, the current date and time will be used (as in the examples above).

The PHP mktime() function returns the Unix timestamp for a date. The Unix timestamp contains the number of seconds between the Unix Epoch (January 1 1970 00:00:00 GMT) and the time specified.

Syntax

Créer une date avec mktime()

Le paramètre facultatif horodatage dans la fonction date() spécifie un horodatage. Si omis, la date et l'heure actuelles seront utilisées (comme dans exemples ci-dessus).

La fonction PHP mktime() renvoie l'horodatage Unix pour une date. L'horodatage Unix contient le nombre de secondes entre l'époque Unix (1er janvier 1970 00:00:00 GMT) et l'heure indiquée.

Syntaxe

mktime(hour, minute, second, month, day, year)

The example below creates a date and time with the date() function from a number of parameters in the mktime() function:

L'exemple ci-dessous crée une date et une heure avec fonction date() à partir d'un certain nombre de paramètres dans le Fonction mktime() :

Example


 <?php
 $d=mktime(11, 14, 54, 8, 12, 2022);
 echo "Created date is " . date("Y-m-d h:i:sa", $d);
 ?>

Create a Date From a String With strtotime()

The PHP strtotime() function is used to convert a human readable date string into a Unix timestamp (the number of seconds since January 1 1970 00:00:00 GMT).

Syntax

Créer une date à partir d'une chaîne avec strtotime()

La fonction PHP strtotime() est utilisée pour convertir un fichier lisible par l'homme chaîne de date dans un Horodatage Unix (le nombre de secondes depuis le 1er janvier 1970 00:00:00 GMT).

Syntaxe

strtotime(time, now)

The example below creates a date and time from the strtotime() function:

L'exemple ci-dessous crée une date et une heure à partir de la fonction strtotime() :

Example

 <?php
 $d=strtotime("10:30pm April 15 2022");
 echo "Created date is " . date("Y-m-d h:i:sa", $d);
  ?>

PHP is quite clever about converting a string to a date, so you can put in various values:

PHP est assez intelligent pour convertir une chaîne en une date, vous pouvez donc y mettre différentes valeurs :

Example

 <?php
 $d=strtotime("tomorrow");
 echo date("Y-m-d h:i:sa", $d) . "<br>";
 $d=strtotime("next Saturday");
 echo date("Y-m-d h:i:sa", $d) . "<br>";
 $d=strtotime("+3 Months");
 echo date("Y-m-d h:i:sa", $d) . "<br>";
?>

However, strtotime() is not perfect, so remember to check the strings you put in there.

Cependant, strtotime() n'est pas parfait, alors n'oubliez pas de vérifier les chaînes que vous y mettez.

More Date Examples

The example below outputs the dates for the next six Saturdays:

Plus d'exemples de dates

L'exemple ci-dessous affiche les dates des six prochains samedis :

Example

 <?php
$startdate = strtotime("Saturday");
$enddate = strtotime("+6 weeks", $startdate);

while ($startdate < $enddate) {
  echo date("M d", $startdate) . "
";
  $startdate = strtotime("+1 week", $startdate);
}
 
 ?>

L'exemple ci-dessous affiche le nombre de jours jusqu'au 4 juillet :

Example

 $d1=strtotime("July 04");
 $d2=ceil(($d1-time())/60/60/24);
